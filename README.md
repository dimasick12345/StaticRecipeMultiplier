# Static Recipe Multiplier
[ENG]
Allows you to increase the cost of crafting by multiplying the recipe price by an integer in all recipes except those that contain any ore or raw material.

This mod was created using reverse vibe coding (I'm too lazy to learn Lua :(). Essentially, it's a result of SuperGrok's work, not my personal knowledge of Lua.
But I don't think that made this mod any worse. It's extremely stable, even in a modpack with a large number of mods.

 # Статический Множитель Рецептов
 [RUS]
Позволяет увеличить стоимость крафта с помощью умножения цены рецепта на целое число во всех рецептах кроме тех где есть любая руда/сырьё.

Этот мод был создан с помощью техники реверсивного вайб кодинга лень учить lua :( , по сути это результат SuperGrok'а, а не мои личные познания в lua. 
Но я думаю это не сделало этот мод хуже. Мод ведет себя максимально стабильно даже в модпаке c большим количеством модов.
